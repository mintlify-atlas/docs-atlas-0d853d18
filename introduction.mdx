## What is ElectroHome?

ElectroHome is a full-featured online store built with Django, designed to sell electronics and home appliances to customers in the Colombian market. It provides a complete shopping experience — from browsing products to completing secure payments via Wompi — while offering store administrators powerful tools to manage inventory, orders, and customers.

## Architecture overview

ElectroHome follows Django's standard MTV (Model-Template-View) pattern:

- **Models** define products, orders, cart items, and user profiles
- **Views** handle request routing, business logic, and context preparation
- **Templates** render HTML for the storefront and admin interfaces
- **Settings** are split across `base.py`, `local.py`, and `production.py` for environment-specific configuration

The production deployment uses environment variables for all secrets and external service credentials, with HTTPS enforced via `SECURE_SSL_REDIRECT`.
