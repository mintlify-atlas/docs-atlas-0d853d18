---
title: "Quickstart"
description: "Get ElectroHome running on your local machine for development."
---

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/your-org/electrohome.git
    cd electrohome
    ```
  </Step>

  <Step title="Create and activate a virtual environment">
    <CodeGroup>
      ```bash macOS / Linux
      python -m venv venv
      source venv/bin/activate
      ```

      ```bash Windows
      python -m venv venv
      venv\Scripts\activate
      ```
    </CodeGroup>
  </Step>

  <Step title="Install dependencies">
    ```bash
    pip install -r requirements.txt
    ```
  </Step>

  <Step title="Run migrations">
    The default development settings (`electrohome.settings`) use SQLite, so no database setup is required.

    ```bash
    python manage.py migrate
    ```
  </Step>

  <Step title="Create a superuser">
    Create an admin account to access the Django admin panel.

    ```bash
    python manage.py createsuperuser
    ```

    Follow the prompts to set a username, email, and password.
  </Step>

  <Step title="Start the development server">
    ```bash
    python manage.py runserver
    ```

    <Note>
      The development server runs at [http://127.0.0.1:8000](http://127.0.0.1:8000). The Django admin panel is available at [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/).
    </Note>
  </Step>
</Steps>

## Next steps

Once you have the local server running, see the [Deployment](./deployment) guide to deploy ElectroHome to Render using the `electrohome.settings.production` settings module.
